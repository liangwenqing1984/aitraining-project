# 数据采集功能

## 功能目标

从智联招聘、前程无忧等招聘平台自动采集职位数据，支持批量任务、断点续传、智能去重、WAF 反爬对抗、实时进度推送。

## 业务流程

1. **创建任务** → 选择平台（智联/前程无忧/全部）→ 输入关键词（点"添加"按钮加入列表，支持多关键词）→ 选择目标城市（多选下拉框）→ 设置最大页数 → 点击"创建"
2. **批量创建** → 输入多关键词 + 多城市 → 系统自动生成笛卡尔积组合任务 → 可预览组合结果 → 确认创建
3. **启动任务** → 任务列表点击"启动" → 状态 `pending → running` → Puppeteer 启动 Headless Chrome → 搜索关键词 → 解析搜索结果列表 → 逐个打开详情页抓取完整信息
4. **实时监控** → WebSocket 推送进度条（`task:progress`）+ 分级彩色日志（`task:log`）+ 状态变更（`task:status`）→ 前端展示运行时长、抓取速率、成功/失败计数
5. **任务完成** → 生成 Excel 文件 → 写入 `csv_files` 表 → 状态变为 `completed` → WebSocket 推送 `task:completed`

## 功能模块

### 页面列表
- **任务列表**：`/crawler` — 分页表格（名称/来源/状态/进度/记录数/操作按钮），支持筛选
- **创建任务**：`/crawler/create` — 关键词输入 + 城市多选 + 页数设置
- **批量创建**：`/crawler/batch` — 多关键词 × 多城市组合预览
- **实时监控**：`/crawler/monitor/:taskId` — 进度条 + 实时日志流 + 状态指示

### 爬虫核心逻辑 (`zhilian.ts` / `job51.ts`)
- 搜索页：构造搜索 URL → 加载页面 → 解析职位卡片列表（多策略 CSS 选择器降级）
- 详情页：逐个打开职位详情 → 提取薪资/公司/技能/学历/经验等字段
- 断点续传：启动时检查已有 Excel → 读取已采集 jobId 集合 → 跳过已完成项
- WAF 对抗：页面加载后检测拦截信号 → 自动降级串行 + 随机延迟 (2-5s) + 指纹随机化
- 智能去重：基于 jobId 去重，新数据增量追加到 Excel

### 任务状态机
```
pending → running → completed
                  → failed (可恢复)
         → paused → running (恢复)
```

## 输入输出

- 输入：
  ```json
  {
    "name": "Java开发-北京",
    "source": "zhilian",
    "config": {
      "keywords": ["Java开发", "Java工程师"],
      "cities": ["北京", "上海"],
      "maxPages": 5,
      "maxConcurrency": 3
    }
  }
  ```
- 输出：Excel 文件（`code/backend/data/csv/job_data_{taskId}.xlsx`），包含 20+ 字段（公司名、职位名、薪资、学历、经验、技能标签、职位描述等）

## 业务规则

- 并发控制：默认并发数 3，遇到 WAF 自动降级为串行（并发数 1）
- 断点续传：每次启动时自动从 Excel 加载已采集的 jobId 集合
- 错误重试：单个详情页抓取失败不中止整个任务，记录 error_count
- 文件命名：`job_data_{taskId}.xlsx`，支持跨会话恢复

## 异常处理

- 浏览器崩溃（`browser disconnected`）：任务标记 `failed`，用户可点击"恢复"从断点继续
- 反爬拦截升级：AI 分类页面类型 → 如果连续检测到验证码/WAF → 自动暂停任务并通知用户
- 选择器失效（网站改版）：多策略降级解析 → 所有策略失败时记录 HTML 快照供诊断
- 任务取消（用户点击"停止"）：通过 AbortSignal 传递取消信号 → 爬虫 `yield` 循环检测并优雅退出
