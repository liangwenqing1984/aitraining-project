# 接口测试

> 目的：验证 47 个 REST API 端点的请求/响应正确性。

## 任务管理 `/api/tasks`

| 编号 | 场景 | 方法 | 路径 | 前置条件 | 预期结果 |
| --- | --- | --- | --- | --- | --- |
| API-01 | 创建任务 | POST | `/api/tasks` | 提供完整 config | 返回 200，data.id 为 UUID |
| API-02 | 创建任务-缺少参数 | POST | `/api/tasks` | config 无 keywords | 返回 400，error 不为空 |
| API-03 | 任务列表 | GET | `/api/tasks?page=1&pageSize=10` | 已有任务 | 返回分页数据，含 total/rows |
| API-04 | 任务详情 | GET | `/api/tasks/:id` | id 存在 | 返回任务完整字段 |
| API-05 | 任务详情-不存在 | GET | `/api/tasks/nonexistent` | - | 返回 404 |
| API-06 | 任务日志 | GET | `/api/tasks/:id/logs` | 任务已运行过 | 返回日志数组，含 timestamp/level/message |
| API-07 | 启动任务 | POST | `/api/tasks/:id/start` | status=pending | 状态变更为 running |
| API-08 | 启动已运行任务 | POST | `/api/tasks/:id/start` | status=running | 返回 error"任务已在运行中" |
| API-09 | 停止任务 | POST | `/api/tasks/:id/stop` | status=running | 浏览器关闭，状态变更 |
| API-10 | 删除任务 | DELETE | `/api/tasks/:id` | 任务存在 | 返回 success，任务及相关联数据删除 |
| API-11 | 省市列表 | GET | `/api/tasks/regions/list` | - | 返回省/城市数组 |

## AI 服务 `/api/llm`

| 编号 | 场景 | 方法 | 路径 | 预期结果 |
| --- | --- | --- | --- | --- |
| API-16 | 增强启动 | POST | `/api/llm/enrich/:taskId` | 返回 { status: "started" }，WebSocket 开始推送进度 |
