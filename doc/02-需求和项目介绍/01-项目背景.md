# 项目背景

## 背景说明

- 本项目用于：构建一个集数据采集、AI 增强、智能分析、自然语言查询于一体的全栈招聘数据平台
- 当前业务背景：招聘市场数据分散在各招聘平台（智联招聘、前程无忧等），原始数据格式混乱（薪资写法多达十几种），手工整理耗时且无法规模化分析
- 期望解决的问题：
  - 自动化采集多平台职位数据，支持多关键词 × 多城市批量任务
  - 通过 AI 大模型将非结构化原始数据标准化（薪资、技能、学历、行业分类）
  - 自动生成深度市场洞察报告，替代人工统计分析
  - 提供自然语言查询界面，降低数据分析门槛
  - 智能对抗招聘平台反爬机制，提升爬虫稳定性

## 培训价值

- 学员可通过本项目学习：
  - Vue 3 Composition API + Element Plus 前端开发
  - Node.js + Express + TypeScript 后端架构
  - PostgreSQL 数据库设计与 SQL 查询优化
  - Puppeteer 浏览器自动化爬虫技术
  - LLM API 集成（OpenAI / DeepSeek / Anthropic / Ollama 本地模型）
  - WebSocket 实时通信 + 任务状态推送
  - OAuth2 授权码模式认证对接
  - AES-256-GCM API Key 加密存储
- 本项目可演示的关键能力：
  - 真实生产级全栈项目的完整代码结构
  - 多 LLM 提供商统一路由与降级策略
  - Prompt Engineering 在结构化数据提取中的应用
  - Text-to-SQL 安全白名单校验方案
  - 反爬 WAF 检测与智能应对策略
