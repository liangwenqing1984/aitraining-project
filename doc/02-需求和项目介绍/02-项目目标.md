# 项目目标

## 总体目标

- 构建"智能招聘数据爬虫与 AI 分析平台"，覆盖从数据采集到智能分析的完整链路
- 集成四大 AI 能力：数据增强标准化、智能市场洞察、自然语言查询、AI 反爬对抗
- 支持多 LLM 提供商（云端 OpenAI/DeepSeek/Anthropic/智谱 + 本地 Ollama），用户可自由配置任务路由
- 提供 OAuth2 认证 + WebSocket 实时推送的完整企业级体验

## 本期 Demo 目标

- 完成 47 个 REST API 端点的开发与联调
- 实现智联招聘 + 前程无忧双平台爬虫
- 落地 6 张数据库表的自动迁移与 ORM 封装
- 交付 4 个前端功能模块：数据采集、智能分析、自然语言查询、AI 配置管理
- 集成文档系统（Docs 页面）供学员查看完整项目说明

## 成功标准

- 能完成核心业务流程演示：创建任务 → 爬取数据 → AI 增强 → 智能分析 → 自然语言查询
- 能在本地成功启动和验证（一键 `start-dev.bat`）
- LLM 增强成功率 ≥ 90%（3 层 JSON 降级解析 + 3 次重试）
- 代码结构和实现方式符合项目规范（TypeScript 严格模式 + ESLint）
- 反爬检测可识别 6 种页面类型（正常/验证码/WAF/登录/错误/空白）
