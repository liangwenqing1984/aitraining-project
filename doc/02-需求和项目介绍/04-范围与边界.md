# 范围与边界

## 本期范围

- 必做功能：
  - 智联招聘、前程无忧双平台数据采集（关键词搜索 + 详情页抓取）
  - 批量任务创建（多关键词 × 多城市笛卡尔积组合）
  - 断点续传（浏览器崩溃后从上次中断位置恢复）
  - AI 数据增强标准化（薪资/技能/学历/行业/福利/工作模式 8 个维度）
  - AI 智能市场洞察报告生成（Markdown + ECharts 配置）
  - 自然语言查询（Text-to-SQL + 安全白名单校验）
  - AI 反爬检测与应对策略建议
  - LLM 配置管理（多提供商 + 任务路由）
  - WebSocket 实时进度推送 + 分级彩色日志
  - OAuth2 登录认证 + Token Cookie 管理
- 必做页面：
  - 数据采集（任务列表/创建/批量创建/实时监控）
  - 智能分析（基础图表 + AI 深度分析报告）
  - 智能查询（对话式 NL 查询界面 + 查询历史）
  - AI 配置（模型配置 + 任务路由 + 连接测试）
  - 文件管理（文件列表/预览/下载/分析入口）
  - 项目文档（层级化交互式文档系统）
- 必做接口：47 个 REST API 端点（任务 11 + 文件 8 + 分析 5 + AI 17 + 认证 6）

## 非本期范围

- 暂不实现：
  - RAG 知识库（向量化职位搜索、语义相似推荐）
  - PDF 报告导出（当前仅前端 Markdown 渲染）
  - 定时任务调度（需手动触发采集和增强）
  - 多租户数据隔离（当前只有一个 schema）
  - 移动端适配
  - 国际化（仅中文界面）
- 仅演示不落地：
  - 验证码图像识别（当前仅检测类型，不做自动破解）
  - 分布式爬虫集群部署

## 边界说明

- 依赖外部系统的部分：
  - OAuth2 认证服务 `leaf-auth-server.dev.jinxin.cloud`（授权码模式）
  - PostgreSQL 数据库 `10.1.1.113:7300`（SeaboxSQL 发行版）
  - LLM API 服务（DeepSeek / OpenAI / Anthropic / 智谱 云端，或 Ollama 本地）
- 可使用 Mock 的部分：
  - 本地 Ollama 不可用时，AI 增强会报错（不支持 Mock 数据）
  - 数据库不可用时，系统无法启动（不支持离线模式）
  - 认证服务不可用时，可跳转到登录页但无法完成认证
