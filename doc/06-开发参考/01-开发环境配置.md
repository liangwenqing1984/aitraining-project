# 开发环境配置

本文档用于说明项目开发前需要准备的基础环境、本机配置、联调资源和常用初始化命令。

## 快速清单

- 安装基础工具：Git、Node.js、包管理器、IDE、数据库客户端
- 获取个人数据库账号、`appCode`、`moduleCode`
- 熟悉 Auth、工作流、调度、文件服务等联调资源
- 完成本地仓库和项目初始化

## 1. 基础环境

建议准备以下环境：

- 操作系统：Windows 10 及以上版本
- Git：2.x 及以上版本
- Node.js：24 及以上版本
- 包管理器：`pnpm` 或 `npm`
- 数据库客户端：DBeaver Community

## 2. IDE 与工具推荐

- Visual Studio Code
- IntelliJ IDEA Community Edition
- DBeaver Community
- Git
