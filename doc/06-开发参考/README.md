# 开发参考

## 目录目的

本目录用于存放开发时需要查阅的外部资料、环境配置说明和本地初始化建议。

## 阅读建议

- 搭建环境时先看 `01-开发环境配置.md`
- 涉及认证时看 `02-Auth服务接口说明.md` 和 `07-OAUTH2授权码模式对接说明.md`
- 涉及基础服务时看 `03-Base服务接口说明.md`
- 涉及数据库能力时看 `04-SeaboxSQL函数和插件说明.md`
- 初始化本地开发时看 `05-AI本地初始化建议.md`

## 使用原则

- 本目录用于提供依据和参考，不替代项目自身需求与设计文档
- 如参考内容和项目设计不一致，应以项目设计文档为准，并记录差异
