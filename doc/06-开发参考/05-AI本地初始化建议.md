# AI本地初始化建议

## 初始化目标

帮助学员或 AI 在本地快速完成项目开发环境准备，并建立基本版本管理习惯。

## 建议检查项

- Git 已安装并可用
- Node.js 版本符合项目要求
- 包管理器已安装
- 必要环境变量已配置
- 项目依赖可正常安装

## 建议命令

```bash
git init
git status
node -v
npm -v
```

## 本地版本管理建议

- 完成一个独立功能后进行一次本地提交
- 重要节点前先查看变更，避免混入无关文件
- 生成文件、缓存文件和临时文件不建议纳入版本管理
- 需要回退时优先使用本地提交历史，而不是直接删除代码
