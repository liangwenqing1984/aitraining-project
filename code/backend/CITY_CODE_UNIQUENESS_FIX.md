## 📋 问题描述

用户反馈："还有不少城市的城市代码用的同一个，请继续修正"

经过检查发现，之前生成的城市代码映射表中存在大量重复代码的问题：
- 同一省份的多个城市使用相同代码（如河北11个城市都用561）
- 不同省份的城市代码冲突（如山西和辽宁的代码重叠）

3. **湖北省 vs 海南省**: 代码736-738重叠

#### 修正1: 山西省与辽宁省分离
