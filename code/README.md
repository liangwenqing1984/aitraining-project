# Code Directory

本目录用于存放 Demo 项目的实际代码与脚本。

## 目录说明

- `code/`：默认作为 Node 服务工程根目录使用
- `sql/`：`SeaboxSQL` 相关脚本目录，如建表脚本、初始化数据脚本、查询示例等
- `test/`：测试目录，可存放测试代码、测试脚本和测试说明

## 使用说明

- 本目录默认用于存放 Node 服务代码
- 如需使用 `src/`，建议放在 `code/src/` 下
- 不要在仓库根目录单独创建 `src/`、`app/`、`server/` 等代码目录
- 学员或 AI 应根据 `doc/` 下的需求、设计和规范生成对应代码
- 数据库统一使用 `SeaboxSQL`
