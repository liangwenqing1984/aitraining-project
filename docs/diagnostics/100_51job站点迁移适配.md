# 51job 站点迁移适配（URL → Vue.js SPA → 数据提取）

## 涉及提交

| 提交 | 修复内容 |
|------|---------|
| `29a31ba` | 适配 51job 新版站点，URL 迁移至 we.51job.com，增强 Vue.js SPA 数据提取 |

## 问题

任务 `5210614d-044d-4f27-98c5-effb0b7a87ae` 使用 51job 爬虫，所有页面返回 39 字节空 HTML：

```
页面 HTML 长度: 39 字符
🤖 AI分类: type=empty, confidence=1.00
本页提取 0 条职位
⚠️ 爬取完成但未采集到任何数据
```

根源：51job 已废弃 `search.51job.com/list/...` 格式，迁移至新域名 `we.51job.com/pc/search?...`。

---

## 根因分析

### 1. URL 格式已废弃

| 对比 | 旧格式 | 新格式 |
|------|--------|--------|
| URL | `search.51job.com/list/010000,...,%E9%94%80%E5%94%AE,2,1.html` | `we.51job.com/pc/search?keyword=销售&city=010000&page=1` |
| 响应 | HTTP 200, 0 字节 | HTTP 200, 8005 字节 |
| 渲染 | 无内容 | `<div id="app"></div>` + 13 个 JS bundle |

### 2. 新站点是 Vue.js SPA

- 初始 HTML 仅 `<div id="app"></div>` 外壳
- 依赖 13 个 `51jobcdn.com` JS bundle 渲染职位列表
- 所有 DOM 选择器（`.j_joblist .e`、`.dw_table .el` 等）在新站点中不存在

### 3. 新站点有更强的反爬

- 阿里云滑块验证（`window.aliyun_captchaid`）
- `acw_sc__v2` cookie
- HMAC-SHA256 签名（API 请求头 `sign`）
- API 端点迁移至 `cupid.51job.com/open/noauth/search-pc`

## 修复

### 1. URL 格式迁移

```typescript
// 修复前
const baseUrl = `https://search.51job.com/list/${cityCode},000000,0000,00,9,99,${keyword},2`;
const url = `${baseUrl},${currentPage}.html`;

// 修复后
const url = `https://we.51job.com/pc/search?keyword=${keyword}&city=${cityCode}&page=${currentPage}&pageSize=20`;
```

### 2. 预热流程增强

两次访问建立完整 cookie 链：

```
we.51job.com/             → 基础 session cookie
we.51job.com/pc/search?keyword=java&city=010000  → acw_sc__v2 反爬 cookie
```

### 3. XHR 拦截捕获 API 响应

浏览器内 JS 自动处理 HMAC 签名，XHR 拦截直接获取签名后的 JSON 响应：

```typescript
// 支持两种 API 端点
const isApiUrl = respUrl.includes('51job.com') && (
  respUrl.includes('/api/') || respUrl.includes('/open/') || respUrl.includes('search-pc')
);
```

支持 7 种响应结构（`resultbody.job.items` / `resultbody.data` / `body.jobList` / `data.results` 等）。

支持 10+ 种字段命名映射（`jobName`/`title`/`positionName`、`companyName`/`coName`/`corpName` 等）。

自动构造详情页 URL：
```
jobs.51job.com/{city}/{jobId}.html
we.51job.com/pc/detail?jobId={encryptJobId}
```

### 4. SPA 感知选择器全面更新

**等待条件**（Vue.js / Element UI / Vant）：
```
[class*="jobItem"], [class*="job-item"], .el-card, #app [class*="result"] [class*="item"]
```

**职位卡片内部提取**（每个字段 5+ fallback）：
```
标题: [class*="title"], [class*="name"], [class*="job"], h3, h2
公司: [class*="company"], [class*="cname"], [class*="corp"], [class*="enterprise"]
薪资: [class*="salary"], [class*="pay"], [class*="sal"], [class*="wage"]
城市: [class*="city"], [class*="area"], [class*="location"], [class*="region"]
```

### 5. 详情页抓取增强

所有选择器升级为新旧站点兼容版本（新增 `[class*="position"] h2`、`[class*="wage"]`、`[class*="enterprise"]` 等）。

### 6. 分页器适配

支持 Element UI / Ant Design 分页器组件检测，兜底在本页 0 条结果时自动停止。
