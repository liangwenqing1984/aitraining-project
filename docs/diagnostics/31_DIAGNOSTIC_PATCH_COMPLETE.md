## 🔍 问题定位能力

通过这个诊断系统，我们可以清楚地分析：

### 1. **哪个策略有效？**
- 如果策略1提取了18个，其他都是0 → 策略1是主要来源
- 如果多个策略都有贡献 → 需要评估是否需要多策略融合

### 2. **为什么只解析到18个？**
- **策略1失败2次**：可能是标题过滤、企业名称提取失败或链接缺失
- **策略2全部失败**：可能是因为与策略1重复而被全局去重过滤
- **策略3全部重复**：说明链接已被策略1/2覆盖

### 3. **优化方向**
- 如果策略1失败原因是"标题过短" → 进一步放宽限制
- 如果策略1失败原因是"缺少链接" → 检查选择器是否正确
- 如果策略2/3都因重复失败 → 考虑移除冗余策略提升性能
